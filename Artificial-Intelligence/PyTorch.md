# Pytorch

## 安装

[教程](https://blog.csdn.net/MCYZSF/article/details/116525159)

+ 新建环境

  ```bash
  conda create -n pytorch python=3.10
  activate pytorch
  ```

+ 安装Pytorch

  进入[pytorch官网](https://pytorch.org/)

  ```bash
  conda install pytorch torchvision torchaudio cpuonly -c pytorch
  ```

  有N卡可以下载cuda
